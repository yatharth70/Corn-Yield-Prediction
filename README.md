# Corn Yield Prediction 🌽

## 📌 Problem
Accurately predicting crop yield is essential for optimizing agricultural planning, resource allocation, and decision-making.

## 💡 Solution
Developed a regression model to predict corn yield based on input features, using polynomial regression to capture non-linear patterns in the data.

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Approach
- Performed data preprocessing and cleaning
- Implemented Linear Regression as baseline model
- Applied Polynomial Feature Transformation
- Trained Polynomial Regression model for better accuracy
- Visualized regression curves to compare model performance

## 📊 Results
- Polynomial regression captured non-linear relationships effectively  
- Improved prediction accuracy compared to linear regression  
- Provided better fit for agricultural yield data trends
